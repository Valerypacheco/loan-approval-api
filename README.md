# 🏦 Loan Approval Prediction System

End-to-end Machine Learning project that predicts whether a loan application will be approved or rejected based on user financial data.

---

## 🚀 Features

- Machine Learning model (Random Forest)
- FastAPI backend for predictions
- Interactive web interface (HTML, CSS, JavaScript)
- Real-time predictions using API calls
- Clean and structured project architecture

---

## 🧠 Model Details

- Algorithm: Random Forest Classifier
- Features:
  - Income
  - Credit Score
  - Loan Amount
  - Debt-to-Income Ratio (DTI)
  - Employment Status
- Output:
  - Loan Approved (1)
  - Loan Rejected (0)

---

## 🛠 Tech Stack

- FastAPI
- Scikit-learn
- Pandas
- HTML / CSS / JavaScript
